# Instant Context Sidebar

A browser extension that uses AI to instantly analyze any webpage and present a clean sidebar with a page summary, key term definitions, related links, and browsing history — all powered by Azure OpenAI.


> **Video Demo:** [Watch on YouTube](https://youtu.be/T_r4rKX_CvU)

---

## Purpose

When browsing the web, you often land on pages full of jargon, dense content, or unfamiliar topics. Instead of opening multiple tabs to look things up, Instant Context Sidebar gives you everything you need in one glance — right beside the page you're reading.

---

## Key Features

### Summary
- One-click "What this page is really about" explainer
- Concise 2–4 sentence summary generated by AI
- Highlighted with an accent bar for quick scanning

### Definitions
- Automatically identifies up to 8 technical terms or jargon on the page
- Provides clear, plain-language definitions for each
- Numbered cards with hover effects for easy browsing

### Related Links
- AI-suggested topics for deeper exploration (opens Google search)
- Also extracts existing links from the page itself
- Grouped into "Suggested Topics" and "Links on This Page"

### History
- Keeps a log of the last 30 analyzed pages
- Each entry shows an AI-generated keyword tag, summary snippet, hostname, and relative timestamp
- Click any history card to revisit the original page
- Clear all history with one click

### UI / UX
- Floating action button (bottom-right) on every page — click to open
- Polished gradient header with light/dark theme toggle
- Smooth staggered animations on all cards and panels
- Keyboard shortcut support: `Ctrl+Shift+S` (Windows/Linux) or `Cmd+Shift+S` (Mac)
- Works on both **Chrome** and **Firefox**

## How to Install

### Chrome
1. Open `chrome://extensions` in your browser
2. Enable **Developer mode** (toggle in the top-right)
3. Click **Load unpacked**
4. Select the `instant-context-sidebar` folder

### Firefox
1. Open `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**
3. Select the `manifest.json` file inside the `instant-context-sidebar` folder

## Configuration

Before using the extension, add your Azure OpenAI credentials to `config.json`:

```json
{
  "azure_openai": {
    "endpoint": "https://YOUR-RESOURCE.openai.azure.com/",
    "deployment_name": "YOUR-DEPLOYMENT-NAME",
    "api_key": "YOUR-API-KEY",
    "api_version": "2024-08-01-preview"
  }
}
```

You can get these values from the [Azure Portal](https://portal.azure.com) under your Azure OpenAI resource.

After updating the file, reload the extension in your browser.

## How to Use

1. **Open any webpage** — you'll see a purple bolt icon in the bottom-right corner
2. **Click the icon** (or press `Ctrl+Shift+S`) to open the sidebar
3. The extension automatically analyzes the page and populates:
   - **Summary** tab — what the page is about
   - **Definitions** tab — key terms explained
   - **Links** tab — related topics and page links
   - **History** tab — previously analyzed pages
4. **Switch themes** using the moon/sun icon in the header
5. **Re-analyze** anytime using the button at the bottom
6. **Close** the sidebar by clicking the X button, the floating icon, or pressing the shortcut again

## Tech Stack

- **Manifest V3** — modern extension format supported by Chrome and Firefox
- **Azure OpenAI (GPT)** — powers summary, definitions, related links, and keyword generation
- **Vanilla JavaScript** — no frameworks, no build step
- **CSS Animations** — staggered card entrances, panel transitions, pulse effects
- **chrome.storage.local** — persists theme preference and browsing history
